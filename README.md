# ⚒️ Development Environment
- **iOS**: 16.0 or later  
- **IDE**: Xcode 16.1  
- **Version Control**: Git  

<br>

# ⚙️ Dependencies / Libraries
- **Frameworks**: SwiftUI, Combine  
- **Networking**: Alamofire  
- **SDKs**: KakaoOpenSDK  
- **Bluetooth**: CoreBluetooth  
- **AI Model Integration**: roboflow-swift
